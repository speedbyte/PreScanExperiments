
% ================================================================================================================
% Follow the next steps to create your own experiment with the Demo_TISAccuracy
% ================================================================================================================

Step 1.	Open the PreScan GUI and open the Demo_TISAccuracy.pex file.

Step 2. click on the Invoke Simulation Run Mode Button and Matlab will be opened.

Step 3. Check if Matlab opened the directory where the experiment was saved. Then open the simulink file
	Demo_TISAccuracy_cs.mdl file using Matlab.

Step 4. Start the Simulink model.

Step 5. See the results in the subsystem Mazda RX8.
